## Ideia

Ideia principal é consegui entender como posso escrever de forma simples e funcional um arquivo md.

Não é preciso ter muitas coisas apenas conseguir escrever de froma simples, sem distrações de onde vou salvar ou algo parecido

Outra coisa importante é conseguir incluir formatações de forma rápida, usando atalhos no teclado.

```js
function test(){
  console.log("ssss")
}
```
